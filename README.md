

<h1 style="font-size: 3em; color: #FF0000;">•  PROYECTOS EN SOLIDITY </h1> 


----------------
En esta carpeta encontraremos varios proyectos de blockchain con lenguaje de programacion solidity. Con el fin de poner a funcionar los software hay que descargar el repositorio.
Luego de descargar git desde la pagina fuente, Se puede descargar estos programas dando click en el boton verde y en dowload o siguiendo los pasos a continuacion.
Los comandos para acceder al repositorio son los siguientes.

```Terminal de comandos
git --version
git init
git clone https://github.com/guevaraStian/Proyectos_Solidity.git
cd Proyectos_Solidity
git push origin master
```
---------

Solidity se ejecuta en su IDE. En RemixIDE se abre la carpeta que se va a ejecutar
